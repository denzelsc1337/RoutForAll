How to use:

	- First of all open "config.csv" in Notepad/Excel and update your Your ID and API Key (Do not remove fist row) that you can get from
	  www.onyxberry.com/services/wapi
	  by creating free account.

	- Run Onyx-Whatsapp-API-Client application (If your antivirus blocks the program, Allow it to run or disable your antivirus)
	- After running new file named "chromedriver.exe" will be created, its important for the Onyx Client Application to
	  communicate with your Google Chorme.
	  
	- If your antivirus blocks the program to run, disable your antivirus or allow the program to run.
